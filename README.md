## What I learned
- Creating first-person weapon animations in Blender
- Exporting and integrating animations into UE5
- Setting up animation blueprints
- Implementing a FPS combat system with features like wall avoidance (which is pretty cool)
- Understanding the overall animation pipeline between Blender and UE5
